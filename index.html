Inspection Rules
& Contraband

Система наград за проверку NPC, включая предметы в багаже

Концепция

Когда игрок заканчивает проверку NPC (допрос, сканирование, решение), система оценивает Inspection Rules — набор правил, заданных дизайнером. Каждое сработавшее правило даёт награду.

Одно правило = условие на тегах + награда. Все совпавшие правила срабатывают аддитивно.
Два типа правил
1

Обычное правило

Без Item.* тегов в условии

Проверяется один раз по clue-тегам NPC.

Clue.Decision.Arrest → +100 CR
2

Per-item правило

Есть Item.* теги в условии

Проверяется для каждого предмета. 3 совпадения = награда ×3.

Clue.Decision.Arrest Item.Type.Pills Item.State.Detected

Система автоматически определяет тип: если в условии есть хотя бы один тег Item.* — правило per-item.

Сборка тегов

Для каждого предмета система объединяет теги из трёх источников и проверяет условие правила.

Синтетические теги

Вычисляются автоматически при финализации. Не ставятся вручную.

Тег Условие Источник
Item.State.Contraband Предмет — контрабанда по правилам текущего дня Теги предмета ∩ active contraband tags
Item.State.Detected Обнаружен рентген-сканером Запись при успешном сканировании
Item.State.Confiscated Игрок забрал из багажа Сравнение: было vs осталось
Item.State.Kept Остался в багаже, ушёл с NPC Предмет в багаже при уходе
Контрабанда зависит от дня. Один предмет может быть контрабандой сегодня и легальным завтра. Item.State.Contraband вычисляется в момент проверки. Не путайте с Item.Type.Contraband — это статический тег «всегда контрабанда».
Порядок событий
01Взаимодействие с NPC
Диалог → clue-теги  /  Сканер → detected  /  Документы → clue-теги
02Решение
Игрок нажимает Permit / Refuse / Arrest
Выучивается Clue.Decision.* — награды ещё не выдаются
03Свободное время
Предметы на столе — можно брать и класть обратно
04NPC уходит
Snapshot: Kept vs Confiscated
05Finalize Inspection
Сборка тегов → проверка правил → выдача наград
Примеры
Базовая награда за арест
require Clue.Decision.Arrest
+100 CR
Без Item.* → один раз.
Бонус за найденную контрабандную таблетку
require Clue.Decision.Arrest Item.Type.Pills Item.State.Contraband Item.State.Detected Item.State.Confiscated
+50 CR
per item
2 таблетки совпали → +100 CR.
Контрабанда ушла необнаруженной
require Clue.Decision.Permit Item.State.Contraband Item.State.Kept
ignore Item.State.Detected
−30 CR
per item
Игрок не нашёл контрабанду, штраф за каждый предмет.
Коррупция: изъял контрабанду, отпустил NPC
require Clue.Decision.Permit Item.State.Contraband Item.State.Confiscated
special
Нашёл, забрал, но NPC отпустил — присвоение.
Воровство: изъял легальный предмет
require Item.State.Confiscated
ignore Item.State.Contraband
penalty
Предмет не контрабанда, но игрок его забрал.
Подсказки
  • Без Item.* тегов в условии — правило срабатывает один раз, только по clue-тегам NPC.
  • С Item.* тегами — правило проверяется для каждого предмета. Укажите Item.State.Confiscated или Item.State.Kept, чтобы различать изъятые и оставленные предметы.
  • Item.State.Contraband зависит от дня. Item.Type.Contraband — статический тег «всегда контрабанда». Это разные вещи.
  • Ignore = «если тег есть — правило НЕ срабатывает». Используйте для исключений.
